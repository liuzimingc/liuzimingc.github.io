语言学完全不完全入门

2024-08-27 次阅读 9 min read

为什么是这个标题呢？

一开始看 Wikibook 然后开写了，结合 Wikipedia。后面发现《语言引论》就顺着补充了很多，而后面都在看《语言引论》，再写不过是再抄一遍罢了。所以就不写了。但还是留个纪念。

主要来源 Wikibook &《语言引论》。

涉及语音学、音系学和形态学。

  • linguistic 语言学。
  • phonetics 语音学，研究声音的产生和感知。
  • articulatory phonetics 发音语音学，研究声音如何从物理的发声器官产生。
  • modality 情态，是语言表达与现实、真理的各种关系的方式。
  • sound 声音 / 语音，就是发出的声音。
  • segment 音段，可以在语音流中以物理或听觉方式识别的离散单元语音。
  • phone 音素，最基本的音节。定义较模糊，在某些情况下与 segment 音段同义（也就是segment 音段 = phone 音素！）
  • IPA 国际音标，一种标准的语音符号系统，用来转录音节。一般使用 // 括起来。一个在线链接：Link。
  • consonant 辅音，在发音过程中由气流通过口腔受部分或完全阻碍发出的音。有 active articulator 主动发音器官和 passive articulator 被动发音器官，发音时从主动发声器官到被动发声器官。辅音可以通过 POA 发音位置（place of articulation）和 MOA 发音方式（manner of articulation）来区分。POA 即在哪里，MOA 即如何。

下面是从 POA 角度对辅音进行的部分分类（为了简略直接辅音 →\to→ 音）。

  • bilabial consonant 双唇音。发音时，双唇紧闭。如英语 but 或 put 中的 /b/ 或 /p/。
  • labiodental consonant 唇齿音。发音时，将上牙贴在下唇上。如英语 fuck 中的 /f/。
  • dental consonant 齿音。发音时，舌尖接触上门牙。如英语 thanks 中的 /θ/。
  • alveolar consonant 齿龈音。发音时，舌尖接触牙龈。如英语 set 中的 /s/。
  • postalveolar consonant 齿龈后音。发音时，舌头位于齿龈后方（从前缀 post- 也看得出来）。如英语 ship 中的 /ʃ/。
  • palatal consonant 腭音。发音时，舌体（中部）抵住硬腭（口腔顶部的中部）。如英语 young 中的 /j/。
  • velar consonant 软腭音。发音时，舌背（后部）抵住软腭（口腔上颚的后部）。如英语 kid 中的 /k/。
  • glottal consonant 声门音。发音时，主要使用声门。如英语 hat 中的 /h/。
  • fricative consonant 擦音。发音时，气流通过两个发音器官靠近而形成的狭窄通道。如英语 set 中的 /s/。
  • doubly articulated consonant 双调音辅音。发音时，有两个主要调音部位、有相同调音方法且同时发出。如英语 what 中的 /w/。

下面是从 VOA 角度对辅音进行的部分分类。

  • voiced 浊音。发音时，声带振动。如英语 zet 中的 z。而英语 set 中的 s 不振动，为 voiceless 清音。
  • nasality 鼻音。发音时，有些声音在气流通过鼻腔时产生。如英语 not 中的 /n/。
  • obstruent 阻音。发音时，阻碍气流。如英语 but 中的 /p/。
  • sonorant 响音。发音时，声道中通过连续、非湍流的气流。响音包括非阻音、鼻音等。如英语 what 中的 w。
  • vowel 元音，在发音过程中由气流通过口腔不受阻碍发出的音。主要有三个区分的变量。

下面是对元音进行的部分分类（可以自己感受变化）。

  • height 高度，发音时舌头的高度。如英语 cat 中的 /æ/ 高度较高，而 feet 中的 /i/ 高度较低。则 /æ/ 就是 open vowel / low vowel 开元音 / 低元音，/i/ 就是 closed vowel / high vowel 闭元音 / 高元音。
  • backness 后倾，发音时舌头的水平位置。如英语 cat 中的 /æ/ 舌头位置较前，而 cot 中的 /a/ 舌头位置较后。则 /æ/ 就是 front vowel 前元音，/a/ 就是 back vowel 后元音。
  • rounding 圆唇，发音时嘴唇的圆润程度。如英语 wood 中的 /u/ 较为圆润，而 feet 中的 /i/ 较为不圆润。则 /u/ 就是 rounded vowel 圆唇元音，/i/ 就是 unrounded vowel 不圆唇元音。
  • semi-vowel / semi-consonant 半元音 / 半辅音（同义词），如英语 what 中的 /w/。
  • acoustic phonetics 声学语音学。研究语音如何操纵声波。比如研究声波的形态。
  • auditory phonetics 听觉语音学。研究语音听觉和语音感知。
  • phonology 音系学。研究不同的音素是如何被串接、组合起来而形成特定的语言，表达出特定的意义。
  • phoneme 音素。可以理解为一组 phone，在一种语言中起着一个单位的作用，并在不同的单词之间形成对比。如英语中 keys 和 school 发音为 [kʰiz] 和 [skulz]。但是发 [kʰ] 或 [k] 不会改变单词的含义，所以这两个音段都属于音素 /k/，表示为 /kiz/ 和 /skulz/。
  • allophone 同位音。一个音素可以表示多于一个音，如果两个 phone 属于同一个音素，它们就是同位音。如英语中的 [pʰ] 和 [p]。
  • complementary distribution 互补分布。一般同位音会以互补分布的方式出现，也就是说，它们会出现于不同的环境中。如英语中 [pʰ] 和 [p]，[pʰ] 一定出现在音节开头，而且很可能出现在后面跟着重读元音的时候，其它情况则出现 [p]。如英语中 sport 和 port。互补分布不会出现在同一环境，所以不会导致差异。
  • contrastive distribution 对比分布。它们可以出现在同一环境中，所以会导致差异。如英语中 pat 和 bat 中的 [pʰ] 和 [p]，它们使含义发生了变化，处于对比分布。
  • minimal pair 最小对，一对只有一个音段不同的单词或短语。这两个不同的音段就处于对比分布中。如英语中的 pen 和 pan 就互为最小对，不同的音段就是 [ɛ] 和 [æ]。
  • free variation 自由变换，指两个有差异的语音出现在相同的情境中、这两者可互相替代使用，而母语者都认为正确。如 tomato 中 a 的不同发音 /ɑ/ 和 /eɪ/。
  • phonotactics 语音组合法。研究语言中的语音组合法则。比如在普通话中，/j/ 后面不能跟 /a/，/e/，/o/ 等。
  • morphophonology 形态音系学。研究音系学和形态学之间的相互关系。
  • morphology 形态学。研究单词的内部结构和形成方式。
  • morpheme 语素。语言中最小的有意义单位（语义单位），语音和语义的任意结合体，不能进一步分析。如普通话中，「我」「犹豫」分别是一个语素。如英语中，antigovernment，可以分成 anti-、govern、-ment 三个语素。而 anti 无法再被分成 an 和 ti——它们「没有意义」，自然 an 和 ti 不是语素。
  • root 词根。具有实际词汇意义的语素。如上文中的 govern。
  • affix 词缀。见黏着语素。如上文中的 anti-（前缀） 和 -ment（后缀）。
  • stem 词干。词根和词缀可以组成词干。词干可以是完整的单词，也可以不是。
  • infix 中缀。英语中几乎没有，最常见的是形容词或副词来加强语气，如 fuckin 插入到 unbelievable 中，得到 un-fuckin-believable 他妈的难以置信。
  • free morpheme 自由语素，可以单独使用，也可以结合。如英语中的 govern。

下面是自由语素的分类。

  • lexical morpheme 词汇语素。表达想表达的信息概念。如名词、动词。具有更多的“扩展性”。
  • functional morpheme 功能语素。是功能词，如冠词、连词。
  • bound morpheme 黏着语素。只作为单词的部分出现，即 affix 词缀（有 prefix 前缀、 suffix 后缀等） 。

下面是黏着语素的分类。

  • derivational morpheme 派生语素。可以改变单词的语义。如英语的词缀 un-。
  • inflectional morpheme 屈折语素。不改变单词的语义，只是引入句法变化。如英语的过去式标志 -ed。一般来讲，英语中的屈折语素位于派生语素之后。

  • cranberry morphemec 蔓越莓语素。一种黏着语素，没有独立的含义和语法功能，但和其它语素结合起来就有了意义。如英语中 hunkleberry，berry 是一个自由语素，但 hunkle 只能和 berry 结合，并得到对应的意义：一种形状小的圆的紫蓝色浆果。

  • 树形图。可以反映词（和句）的形态组成。如 unsystematic 的组成（通过下图可以知道不是通过 unsystem 加上 atic 构成的）

  • accidental gap 偶然缺位。指词、语素等理论上符合语法规则，但实际中并不存在。如 disobvious，是可能存在的，但在实际中没有使用过。

  • zero derivation 零派生。这是一种观点，语素连接了一个“空语素”，使语素词性发生变化。如英语中 clean adj. 干净的 →\to→ clean v. 打扫。

  • derivational morphology 派生形态学。研究单词派生相关。

  • inflectional morphology 屈折形态学。研究单词屈折相关。

  • concatenative morphology 连接形态学。研究语素连接，形成单词的过程。

  • nonconcatenative morphology 非连接形态学。研究不涉及词素连接而形成单词的过程，语素一般会通过 reduplication 重复、clipping / truncation 截取、ablaut 元音变化等方式修改。如英语中一般通过加上 -s 标记复数，涉及连接形态学，但在 foot 到 feet 这样的变化中，涉及非连接形态学。

  • allomorph 语素变体。指发音和拼写发生变化，但意义不变的同一个语素。如英语中 plural allomorph 复数语素变体有 [-s], [-z], and [-əz]，类比 caps、bags、buses。
