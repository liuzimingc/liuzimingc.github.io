直陈式现在时

直接搬 cnblogs 的了。

-er

e.g. parler。

  • je parle (-e)
  • tu parles (-es)
  • il parle (-e)
  • nous parlons (-ons)
  • vous parlez (-ez)
  • ils parlent (-ent)

以 -cer 结尾的。e.g. avancer。

  • nous avançons

以 -ger 结尾的。e.g. manger。

  • nous mangeons

e.g. finir，choisir。

  • je finis (-is)
  • tu finis (-is)
  • il finit (-it)
  • nous finissons (-issons)
  • vous finissez (-issez)
  • ils finissent (-issent)

副动词

虽然说不是时态还是写一下，en + 现在分词，即动词现在时复数第一人称去掉 -ons 加上 -ant。

例外：

  • nous avons en ayant
  • nous somme en étant
  • nous savons en sachant

和英语 ing 挺像的。。。

直陈式复合过去时

应该是英语的一般过去 / 现在完成？

  1. 表示在一个近期或遥远的过去完成的动作，动作造成的结果或感受可能与现在仍有关联。
  2. 表示在过去不确定的时刻完成的动作。
  3. 表示过去动作的重复性或在一定期限内延续。

总之就很完成吧。

最近过去时

venir de faire。就是刚刚发生的事情。

反正就变变 venir 以及 de 可能缩合，trés simple。这点挺好的，赞赞。

最近将来时

aller faire。就是英语 be going to do。

直陈式未完成过去时

用来表示过去正在进行的状态或是未完成的动作，动作或状态的起始和结束时间都不明确。

  1. 表示过去处于延续状态的动作，其开始和结束的时间都不明确。
  2. 表示过去重复发生的或是具有习惯性的动作。
  3. 用来描写故事的背景、人物状态、环境、气氛等。
  4. 表示和另一过去动作同时发生的动作。

一般就是直陈式现在时复数第一人称（nous）的词根 + -ais / -ais / -ait / -ions / -iez / -aient。

直陈式简单将来时

可以理解为 will do？？？

词根 + -ai / -as / -a / -ons / -ez / -ont。

一般词根是不定式，例外如：

  • avoir aur
  • être ser
  • aller ir（世界语 iri）
  • faire fer（记下雨那一课的 fera）
  • devoir devr
  • pouvoir pourr
  • savoir saur
  • venir viendr
  • voir verr
  • voulour voudr（记条件式现在时的 je voudrais）
  • pleuvoir pleuvr

条件式现在时

接续上面两个就是小 case。用在独立句基本就是表什么礼貌邀请建议可能发生感慨啥的，有点像 would。

直陈式简单将来时的词根 + 直陈式未完成过去时的词尾（缝合怪）。

  • j'aimerais
  • tu aimerais
  • il aimerait
  • nous aimeriont
  • vous aimeriez
  • ils aimeraient

虚拟式现在时

表意愿，必要性，怀疑，情感或赞赏。

很奇怪，要分两种。对于单数人称和复数第三人称（je、tu、il、ils）用直陈式现在时的复数第三人称（ils）变位词根加上 -e、-es、-e、-ent（即直陈式现在时）。对于剩下的（nous、vous）用直陈式现在时的复数第一人称（nous）变位词根加上 -ions、-iez（即直陈式未完成过去时）。而且往往前面都有 que。

  • que je parle
  • que tu parles
  • que il parle
  • que ils parlent
  • que nous parlions
  • que vous parliez
