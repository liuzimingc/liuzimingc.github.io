尝试取一个抽象标题。

绝对路径：形如 ~/下载/code；相对路径：形如 ./1.cpp。

相对路径顾名思义就是相对的，假设当前是 ~/下载/code，那么其 ./1.cpp 就是 ~/下载/code/1.cpp。What does . refer to?

然后就是非常有用的终端命令！Ctrl + Alt + T 唤出终端。

  • cd 可以接绝对路径也可以接相对路径。如果写 cd .. 即返回上一级，假设当前是 ~/下载/code，执行 cd .. 后就是 ~/下载。cd - 则切换到上一次的目录命令。change directory。
  • mkdir 新建文件夹，如 mkdir fuck。make directory。
  • ls 列出当前所在位置的所有文件。ls --help 各种各样的命令。list。
  • touch 新建文件，如 touch 1.cpp。不是很懂为什么是 touch。
  • cat 显示文件内容，如 cat 1.cpp。concatenate。
  • clear 清屏。
  • ulimit 如 ulimit -s unlimited 就是开无限栈。每次都会重置，所以每次都要输。（输入 ulimit -s 0 可以导致神秘东西。）
  • g++ 美丽的编译。使用 g++ -std=c++14 -O2 -Wall 1.cpp -o 1 可以从 1.cpp 编译出 1 可执行文件。
  • pwd 显示当前路径。print working directory。
  • diff 比较文件。如 diff 1.out 1.ans。

如果要运行一个可执行文件的话，Linux 下可执行文件没有后缀 exe，如运行当前路径下 1 这个可执行文件直接 ./1 即可。

VS Code 终端是 Ctrl + `（经帅气罗阿姨指正🤔），用法一样，注意路径。

最后一些杂项。

  • Firefox 新建标签页 Ctrl + T。
  • 调中文输入法右上角，然后 Shift 切中英文。
  • 按 Tab 可以自动补全文件名。
